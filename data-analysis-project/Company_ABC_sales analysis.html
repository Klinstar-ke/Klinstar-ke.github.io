Page 1: Scope of the analysis

  • Introduction

    • This analysis represents a sales dataset of company ABC with details of sales recorded in each and every day through out the year 2023 and beginning of the year 2024. This dataset contains information about the entries recorded such as Product_ID, Sale_Date, Sales_Rep, Region, Sales_Amount, Quantity_Sold, Product_Category, Unit_Cost, Unit_Price, Customer_Type, Discount, Payment_Method, Sales_Channel, Region_and_Sales_Rep. The data can be accessed through the link given below: Download the Dataset(CSV)

    The research focuses on examining the sales perfomance and trends inorder to help the company make proper insights about the company operations and how to make proper business decisions based on the dataset.

    Data collection and Data processing

    The data set contains 1000 rows and 14 columns of data. We began by first cleaning the dataset and removing the duplicate values and standardizing the date formats using Power query and excel inorder to make it ready for analysis. We didnt have a lot of missing values and hence for the columns with the missing values we treted them using the statistical formulars like mean, mode and median. After data was validated we went into the analytical part where we used Excel tool in the drafting of the pivot tables and later went in to the visualization part.
